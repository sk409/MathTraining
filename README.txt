各フォルダの説明

・JTPC
Unityプロジェクトが格納されているフォルダです

・Instructions
ゲームの説明に関するファイルが格納されているフォルダです。
このフォルダの中にあるinstructions.htmlからゲームの説明を確認することができます。
このinstructions.htmlはChromeのフルスクリーンでの閲覧を想定しています。
instructions.htmlの他にothersというフォルダがありますが、これはinstructions.htmlで使用している素材を格納しています。

・Game
ゲームの実行環境が格納されているフォルダです。
このフォルダの中のJTPC.exeからゲームをプレイすることができます。
